# neovim
Personal setup for neovim

## Requirements

- brew install ctags
- brew install ripgrep
- brew install fzf

## Setup for Mac/Ubuntu usage.

Setup for a dev env for:

1. PHP
2. PRISMA.IO
3. NODE
4. VUE
5. NextJs
6. ReactJs
7. Laravel
